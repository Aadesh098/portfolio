# Avs Portfolio

**A performance-driven, immersive portfolio engineered to showcase modern software craftsmanship.**

This portfolio is a high-performance, visually dynamic platform designed to present projects, skills, and experience through interactive storytelling. It blends 3D visuals, smooth animations, and responsive design to deliver a polished, engaging developer identity.

---

## Core Highlights

- **Interactive UI:** Smooth, animation-rich interface enhancing user engagement  
- **3D Experience:** Integrates Three.js-based visuals for immersive storytelling  
- **Motion Design:** Fluid transitions powered by Framer Motion  
- **Responsive Layout:** Fully optimized across devices and screen sizes  
- **Theme System:** Dynamic light/dark mode with seamless switching  
- **Project Showcase:** Structured display of work with clean navigation  
- **Contact Flow:** Integrated EmailJS for direct communication  
- **Performance Focus:** Optimized rendering and asset handling for speed  
- **Component Design:** Modular, reusable UI architecture  
- **Visual Identity:** Minimal, modern design language with strong aesthetics  

---

## Tech Stack

**Core:** Next.js 14, React 18, TypeScript  
**3D & Graphics:** Three.js (React Three Fiber, Drei, Three Globe)  
**Animations:** Framer Motion, CSS Animate  
**Styling:** Tailwind CSS, Tailwind Merge  
**Utilities:** clsx, mini-svg-data-uri  
**Integrations:** EmailJS  
**UI Enhancements:** React Icons, Lottie  

---

## Vision

This portfolio is built to go beyond static presentation-positioning development as an interactive experience where design, performance, and engineering converge.
